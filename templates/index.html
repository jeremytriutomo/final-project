Dago Coffee – Production & Batch Subsystem

Simple UI to interact with the /inform-order and /trigger-batch APIs of the Production & Batch Subsystem (port 5001).

1. Inform Order POST /inform-order

Send example order data to this subsystem (for integration testing with Sales).

Response

2. Trigger Weekly Batch POST /trigger-batch

Calls /trigger-batch which fetches weekly orders from http://localhost:5000/orders/weekly, aggregates items, computes ingredient usage, logs to SQLite, and updates inventory.
